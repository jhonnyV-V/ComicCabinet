<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

- flutter.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jhonnyV-V/ComicCabinet.git
   ```
3. Install NPM packages
   ```sh
   flutter pub get
   ```
4. Create a .env file following the .env.example template and place your comicvine api there
5. Execute
   ```sh
   flutter pub run build_runner build
   ```
